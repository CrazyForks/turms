    具体状态码细节可查阅im.turms.common.TurmsStatusCode类。

    # 类别

    为方便记忆，Turms的业务状态码（Business status code）参考HTTP状态码设计：

    • 2XXX。表明请求成功
    • 3XXX。表明需要客户端进行重定向操作
    • 4XXX。表明客户端错误
    • 5XXX。表明服务端错误
